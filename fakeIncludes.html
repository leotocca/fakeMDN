fakeIncludes()

Home

La función fakeIncludes() determina si una matriz incluye un determinado elemento, devuelve true o false

Sintaxis

fakeIncludes(array, searchElement);

Parametros

array

Array a iterar en la búsqueda del value.

searchElement

El elemento a buscar.

Valor de retorno

Un boolean.

Ejemplos

fakeIncludes([1, 2, 3],2); // true
fakeIncludes([1, 2, 3],4); // false
fakeIncludes([1, 2, 3],3); // false
fakeIncludes([1, 2, 3],3); // true
fakeIncludes([1, 2, NaN],NaN); // true
