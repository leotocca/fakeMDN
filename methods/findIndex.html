_findIndex

Home

La funcion _findIndex() ejecuta la función callback que se pasa como parametro una vez por cada elemento del array, tambien provisto, hasta que se verifique la condición de la función callback. En ese caso, devuelve el indice del elemento que la verifica. De lo contrario devuelve -1.

Sintaxis

              
                _findIndex(arr, callbackfunction){

                //tu iteración

                });

Parametros

Array

Array que sera iterado

Función

Función que será ejecutada por cada elemento iterado

Descripción

_findIndex() ejecuta la funcion callbackfunction que se pasa como parametro una vez por cada elemento del array, tambien provisto, hasta que se verifique la condición de la función callback. En ese caso, devuelve el indice del elemento que la verifica. De lo contrario devuelve -1.

Ejemplos

Mostrar cada elemento del array 

              
                const fruits = ["apple", "banana", "cantaloupe", "blueberries", "grapefruit"];
                console.log(_findIndex(arr, fruit => fruit === "blueberries"); 
                // output esperado = 3
