_max()

Home

La función _max() devuelve el elemento máximo de un arreglo, acepta arreglos numéricos y de strings.

Sintaxis

              
                const valorMaximo = array._max();

Valor de retorno

El mayor elemento del arreglo

Ejemplos

Obtener el mayor elemento de un array 

              
                arreglo=[1, 2, 3, 4]; 
                valorMaximo = _max.(arreglo); // 4

Ultima palabra en orden alfabético 

              
                arreglo = ["papa", "puerro", "batata", "cebolla"]; 
                ultimaPalabra = _max.(arreglo); // "puerro"
