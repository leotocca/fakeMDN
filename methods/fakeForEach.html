La funcion fakeForEach() ejecuta la función pasada por un parametro una vez, para cada elemento del array provisto

Sintaxis

fakeForEach(arr,fn){
//tu iteración
});

Parametros

Array

Array que sera iterada

Función

Función que será ejecutada por cada elemento iterado

Valor de retorno

undefined

Descripción

fakeForEach() ejecuta la funcion fn una vez por cada elemento de la array. A diferencia de map() o reduce() este siempre devuelve el valor undefined y no es encadenable

Ejemplos

Mostrar cada elemento del array

let arr = ['a', 'b', 'c'];
console.log(fakeForEach(arr, (showEachElement = arr => console.log(arr))));
// output 'a'
// output 'b'
// output 'c'

Convertir un for loop en fakeForEach

const items = ['item1', 'item2', 'item3'];
const copy = [];

// before
for (let i=0; i< items.length; i++) {
copy.push(items[i]);
}

// after
fakeForEach(items, (convertir = arr => copy.push(arr)));
