fakeMin()

Home

La función fakeMin() devuelve el menor elemento de un arreglo, acepta arreglos numéricos y de strings.

Sintaxis

              
                const valorMinimo = fakeMin(arreglo);

Parámetros

arreglo

El arreglo del cual queremos obtener el mayor elemento

Valor de retorno

El menor elemento del arreglo

Ejemplos

Obtener el menor elemento de un array 

              
                arreglo = [1, 2, 3, 4]; 
                valorMinimo = fakeMin(arreglo);   // 1

Primera palabra en orden alfabético 

              
                arreglo = ["papa", "puerro", "batata", "cebolla"]; 
                primeraPalabra = fakeMin(arreglo);   // "batata"
