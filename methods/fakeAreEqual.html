fakeAreEqual

Home

La funcion fakeAreEqual() evalua dos arrays que son pasados como parametro a la función y si es que ambos son identicos (mismos valores en los mismos indices) devuelve "true", sino devuelve "false".

Sintaxis

              
                fakeAreEqual(array1, array2){

                //tu iteración

                });

Parametros

Array1

Array que sera evaluado

Array2

Array que sera evaluado

Descripción

fakeAreEqual() evalua dos arrays que son pasados como parametro a la función y si es que ambos son identicos (mismos valores en los mismos indices) devuelve "true", sino devuelve "false".

Ejemplos

Comparar dos arrays 

              
                const arr1 = ['a', 'b', 'c'];
                const arr2 = ['a', 'b', 'c'];
                console.log(fakeAreEqual(arr1, arr2));
                // output esperado = true;


                const fruitsArr1 = ['orange', 'pear', 'banana'];
                const fruitsArr1 = ['orange', 'banana', 'pear'];
                console.log(fakeAreEqual(arr1, arr2));
                // output esperado = false;
