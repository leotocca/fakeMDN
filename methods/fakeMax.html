fakeMax()

Home

La función fakeMax() devuelve el elemento máximo de un arreglo, acepta arreglos numéricos y de strings.

Sintaxis

const valorMaximo = fakeMax(arreglo);

Parametros

arreglo

El arreglo del cual queremos obtener el mayor elemento

Valor de retorno

El mayor elemento del arreglo

Ejemplos

Obtener el mayor elemento de un array 

                    arreglo=[1, 2, 3, 4]; 
                    valorMaximo = fakeMax(arreglo); // 4

Ultima palabra en orden alfabético 

                  arreglo = ["papa", "puerro", "batata", "cebolla"]; 
                  ultimaPalabra = fakeMax(arreglo); // "puerro"
