fakeIndexOfRecursive()

Home

La funcion fakeIndexOfRecursive() retorna el primer índice en el que se puede encontrar un elemento dado en el array, ó retorna -1 si el elemento no esta presente.

Sintaxis

              
                fakeIndexOfRecursive(array, searchElement)

Parametros

array

Array a iterar.

searchElement

Elemento a encontrar en el array.

Valor de retorno

El índice correspondiente a la primera aparición del elemento en la matriz; -1 si no se encuentra.

Descripción

fakeIndexOfRecursive() compara searchElement con los elementos del array usando igualdad estricta (el mismo método que cuando se usa ===, o el operador igualdad-triple).

Ejemplos

Usando fakeIndexOfRecursive()

El siguiente ejemplo usa fakeIndexOfRecursive() para localizar valores en un array 

              
                var array = [2, 9, 9];
                array.fakeIndexOfRecursive(2); // 0
                array.fakeIndexOfRecursive(7); // -1 
                array.fakeIndexOfRecursive(9, 2); // 2 
                array.fakeIndexOfRecursive(2, -1); // -1 
                array.fakeIndexOfRecursive(2, -3); // 0

Encontrar todas las apariciones de un elemento 

              
                const indices = []; 
                const array = ['a', 'b', 'a', 'c', 'a', 'd'];
                const element = 'a';
                let idx = array.fakeIndexOfRecursive(element);
                while (idx != -1) {
                indices.push(idx);
                idx = array.fakeIndexOfRecursive(element, idx + 1);
                } 
                console.log(indices);
                // [0, 2, 4]
