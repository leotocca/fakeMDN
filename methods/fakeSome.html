La funcion fakeSome() iterara el array y cuando se cumpla la condicion establecida, se ejecutara la funcion

Sintaxis

function fakeSome(arr, fn) {
for (let element of arr) {
if (fn(element)) {
return true;
}
}
return false;
}

Parametros

Array

Array a iterar.

Función

Funcion que se ejecutara si se cumple la condicion

Valor de retorno

Boolean

Descripción

fakeSome() ejecuta fn una vez por cada elemento presente en el array hasta que tome uno con valor true; sino retornara el valor false.

Ejemplo

Mostrar cada elemento del array

const array = [1, 2, 3, 4, 5];
const even = function(element) {
return element % 2 === 0;
}
console.log(fakeSome(array, even));
