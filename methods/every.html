_every()

Home

La funcion _every() devuelve un booleano, true si todos los elementos en el arreglo pasan la condición implementada por la función dada y false si alguno no la cumple.

Sintaxis

            
              _every(array, callbackFunction);

Parametros

Array

Array que sera iterada

callbackFunction

Función para probar cada elemento del array

Valor de retorno

Un boolean.

Descripción

_every() ejecuta la funcion fn una vez por cada elemento de la array hasta encontrar uno que haga retornar un valor false. Si no se encuentra tal elemento, la funcion _Every() de inmediato retorna false. O si fn() retorna verdadero para todos los elementos, every retornará true.

Ejemplo

Todos mayor a 10 

              
                function isBigEnough(element, index, array) {
                  return element >= 10;
                }
                _every([12, 5, 8, 130, 44],isBigEnough);   // false
                _every([12, 54, 18, 130, 44],isBigEnough); // true
