fakeReduce()

Home

La función fakeReduce() aplica una función a un acumulador y a cada valor de un array (de izquierda a derecha) para reducirlo a un único valor.

Sintaxis

let result = fakeReduce(array, reducFunction);

Parametros

array

El array a ser procesado

reducFunction

Es la función que se utilizará para procesar arreglo, puede ser anónima, recibe dos parámetros y devuelve un resultado:
let reducFunction = (param1, param2) => result;

param1, param2

Los dos parámetros que utilizará funcionReductora para calcular el resultado de la función

Valor de retorno

El resultado de reducir el array

Ejemplos

Sumar todos los elementos de un array

reducFunction = (x, y) => x + y;
arrReduced = fakeReduce(array, reducFunction);

Suma de los cuadrados

arrReduced = fakeReduce(array, function (x,y) { return x + y**2; })
