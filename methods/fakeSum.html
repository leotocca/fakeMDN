fakeSum()

Home

La funcion fakeSum() suma todos los elementos de un arreglo y devuelve el valor de esa suma.

Sintaxis

let variable = fakeSum(array);

Parametros

array

Array cuyos elementos queremos sumar

Ejemplo

Sumar los elementos de un array

Calcular la suma de los elementos del vector array y guardarla en la variable suma

let variable = fakeSum(array);
