_union()

Home

La función _union() devuelve un array nuevo que incluye todos los valores de los arrays que se le pasen como argumentos sin repetirlos.

Sintaxis

              
                  _union(arrays);

Parametros

array

Arrays que se van a evaluar para generar un nuevo array que incluya todos sus valores

Valor de retorno

Un array

Ejemplos

              
                  _union([1 ,2], [3, 4], [5, 8], [743, 95], [85, 863, 9]); // [1, 2, 3, 4, 5, 8, 743, 95, 85, 863, 9]

                  _union([1 ,2], [3, 4], [743, 95], [85, 863, 9]); // [1, 2, 3, 4, 743, 95, 85, 863, 9]

                  _union([1 ,2], [3, 4], [85, 863, 9]); // [1, 2, 3, 4, 85, 863, 9]
