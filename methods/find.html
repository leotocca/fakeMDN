_find()

Home

La funcion _find() devuelve el valor del primer elemento del array que cumple la función fn de prueba proporcionada. En cualquier otro caso se devuelve undefined.

Sintaxis

              
                _find(arr,fn){
                //tu iteración
                });

Parametros

Array

Array que sera iterada

Función

Función como condiciona cumplir por algun elemento del array

Valor de retorno

Primer elemento del array que cumpla la condicion o undefined

Descripción

_find() ejecuta la funcion fn El método find ejecuta la función fn una vez por cada índice del array hasta que encuentre uno en el que la fn devuelva un valor true. Si dicho elemento se encuentra, find devuelve inmediatamente el valor del elemento. En caso contrario, find devuelve undefined.

Ejemplos

Demo 

              
                let array1 = [5, 12, 8, 130, 44];
                const lessThanTen = function(element) {
                  return element < 1;
                };
              
                console.log(_Find(array1, lessThanTen));

Encontrar un objeto en un array por una de sus propiedades 

              
                let inventario = [
                { nombre: 'manzanas', cantidad: 2 },
                { nombre: 'bananas', cantidad: 0 },
                { nombre: 'cerezas', cantidad: 5 }
                ];
