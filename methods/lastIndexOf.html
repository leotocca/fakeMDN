_lastIndexOf

Home

La funcion _lastIndexOf() devuelve la ultima posición en el array del valor que se le provee como parametro.

Sintaxis

						
							_lastIndexOf(array, value){

							//tu iteración

							});

Parametros

Array

Array que se evaluara para encontrar el valor.

Valor

Valor que se va a buscar dentro del array provisto como parametro.

Descripción

_lastIndexOf() devuelve la última posición del valor que se provee como parametro dentro del array que tambien se provee como parametro. Si el valor no es encontrado dentro del array, se devuelve el valor -1.

Ejemplos

Encontrar un valor 

							
								const animals = ['Dodo', 'Tiger', 'Penguin', 'Dodo']
								console.log(_lastIndexOf(animals, "Dodo"); 
								// output esperado = 3;
