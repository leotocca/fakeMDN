_isEqual

Home

La funcion _isEqual() evalua dos arrays que son pasados como parametro a la función y si es que ambos son identicos (mismos valores en los mismos indices) devuelve "true", sino devuelve "false".

Sintaxis

              
                array1._isEqual(array2){

                //tu iteración

                });

Parametro

Array2

Array que sera evaluado

Descripción

_isEqual() evalua dos arrays que son y si ambos son identicos (mismos valores en los mismos indices) devuelve "true", sino devuelve "false".

Ejemplos

Comparar dos arrays 

              
                const arr1 = ['a', 'b', 'c'];
                const arr2 = ['a', 'b', 'c'];
                console.log(arr1._isEqual(arr2));
                // output esperado = true;


                const fruitsArr1 = ['orange', 'pear', 'banana'];
                const fruitsArr2 = ['orange', 'banana', 'pear'];
                console.log(fruitsArr1._isEqual(fruitsArr2));
                // output esperado = false;
