fakeIntersection()

Home

La función fakeIntersection() crea un nuevo array con los valores que se repitan en los dos arrays provistos como parametros.

Sintaxis

function fakeIntersection(arr1, arr2){
let newArr = [];
for(let i = 0; i < arr1.length; i++){
for(let j = 0; j < arr2.length; j++){
if(arr1[i] === arr2[j]){
newArr.push(arr2[j]);
};
};
};
return newArr;
};

Parametros

Arrays

arr1 y arr2 - Arrays que se van a comparar

Valor de retorno

Array

Descripción

fakeIntersection() devuelve un array que contiene los valores que se repitan en ambos arrays que se pasan como parametros a la función.

Ejemplo
