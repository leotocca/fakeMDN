fakeIntersection()

Home

La función fakeIntersection() devuelve un array nuevo que incluye todos los valores que se repitan en los arrays que se pasaron como argumentos.

Sintaxis

fakeIntersection(arr1, arr2);

Parametros

array

Arrays que se van a evaluar para generar un nuevo array que incluya los valores que se repiten

Valor de retorno

Un array

Ejemplos

fakeIntersection([1, 2, 3, "a"], ["a", 1, 3, 5, 6]); // [1, 3, "a"]
fakeIntersection([68, 2, true, "a"], ["a", 1, 3, true, 68]); // [68, true, "a"]
fakeIntersection([false, 68, 2, "argh", true, "a"], ["argh", false, 68]); // [false, 68, "argh"]
