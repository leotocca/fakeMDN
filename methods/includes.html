_includes()

Home

La función _includes() determina si una matriz incluye un determinado elemento, devuelve true o false según corresponda.

Sintaxis

              
                  _includes(array, searchElement);

Parametros

array

Array a iterar en la búsqueda

searchElement

Elemento buscado dentro del array

Valor de retorno

Un boolean

Ejemplos

              
                  _includes([1, 2, 3],2); // true
                  _includes([1, 2, 3],4); // false
                  _includes([1, 2, 3],3); // false
                  _includes([1, 2, 3],3); // true
                  _includes([1, 2, NaN],NaN); // true
