_includes()

Home

El método _includes() determina si una matriz incluye un determinado elemento, devuelve true o false según corresponda.

Sintaxis

              
                  array._includes(searchElement);

Parametros

searchElement

Elemento buscado dentro del array

Valor de retorno

Un boolean

Ejemplos

              
                [1, 2, 3]._includes(,2); // true
                [1, 2, 3]._includes(4); // false
                [1, 2, 3]._includes(3); // false
                [1, 2, 3]._includes(3); // true
                [1, 2, NaN]._includes(NaN); // true
