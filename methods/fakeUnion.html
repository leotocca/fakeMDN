fakeUnion()

Home

La función fakeUnion() devuelve un array nuevo que incluye todos los valores de los arrays que se le pasen como argumentos sin repetirlos.

Sintaxis

fakeUnion(arrays);

Parametros

array

Arrays que se van a evaluar para generar un nuevo array que incluya todos sus valores

Valor de retorno

Un array

Ejemplos

fakeUnion([1 ,2], [3, 4], [5, 8], [743, 95], [85, 863, 9]); // [1, 2, 3, 4, 5, 8, 743, 95, 85, 863, 9]
fakeUnion([1 ,2], [3, 4], [743, 95], [85, 863, 9]); // [1, 2, 3, 4, 743, 95, 85, 863, 9]
fakeUnion([1 ,2], [3, 4], [85, 863, 9]); // [1, 2, 3, 4, 85, 863, 9]
