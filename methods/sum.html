_sum()

Home

La funcion _sum() suma todos los elementos de un arreglo y devuelve el valor de esa suma.

Sintaxis

              
                let variable = _sum(array);

Parametros

array

Array cuyos elementos queremos sumar

Valor de retorno

Retorna un integer con la suma del array.

Ejemplo

Sumar los elementos de un array

Calcular la suma de los elementos del vector array y guardarla en la variable suma 

              
                let variable = _sum(array);
