La funcion fakeEvery() devuelve un booleano, true si todos los elementos en el arreglo pasan la condición implementada por la función dada y false si alguno no la cumple.

Sintaxis

fakeForEach(arr,fn){
//tu iteración
});

Parametros

Array

Array que sera iterada

Función

Función para probar cada elemento del array

Valor de retorno

boolean

Descripción

fakeEvery() ejecuta la funcion fn una vez por cada elemento de la array hasta encontrar uno que haga retornar un valor false. Si no se encuentra tal elemento, la funcion fakeEvery() de inmediato retorna false. O si fn() retorna verdadero para todos los elementos, every retornará true.

Ejemplo

Javascript demo

let array1 = [1, 30, 59, 9, 10, 13];
function isBelowThreshold(currentValue) {
return currentValue < 40;
}

function fakeEvery(arr, fn) {
let allPassTheTest = true;
fakeForEach(arr, element => {
if (!fn(element)) {
allPassTheTest = false;
}
});
return allPassTheTest;
}

console.log(fakeEvery(array1, isBelowThreshold));
