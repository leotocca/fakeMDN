_some()

Home

La funcion _some() comprueba si al menos un elemento del array cumple con la condición implementada por la función proporcionada

Sintaxis

            
              function _some(arr, fn) {
                for (let element of arr) {
                  if (fn(element)) { 
                    return true;
                  }
                }
                return false;
              }

Parametros

Array

Array a iterar.

Función

Funcion con la condicion a cumplir

Valor de retorno

Un Boolean

Descripción

_some() ejecuta fn una vez por cada elemento presente en el array hasta que tome uno con valor true; sino retornara el valor false.

Ejemplo

Mostrar cada elemento del array 

            
              const array = [1, 2, 3, 4, 5];
              const even = function(element) { 
                return element % 2 === 0; 
                }
              console.log(_some(array, even));
