_min()

Home

La función _min() devuelve el menor elemento de un arreglo, acepta arreglos numéricos y de strings.

Sintaxis

              
                const valorMinimo = _min(arreglo);

Parámetros

arreglo

El arreglo del cual queremos obtener el mayor elemento

Valor de retorno

El menor elemento del arreglo

Ejemplos

Obtener el menor elemento de un array 

              
                arreglo = [1, 2, 3, 4]; 
                valorMinimo = _min(arreglo);   // 1

Primera palabra en orden alfabético 

              
                arreglo = ["papa", "puerro", "batata", "cebolla"]; 
                primeraPalabra = _min(arreglo);   // "batata"
