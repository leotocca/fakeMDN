La funcion fakeForEach() ejecuta la función pasada como parametro por cada elemento del array provisto

Sintaxis

fakeForEach(arr,fn){
//tu iteración
});

Parametros

Array

Array que será procesada

Función

Función que será ejecutada por cada elemento iterado

Valor de retorno

undefined

Ejemplos

Concatenar a cada elemento del string

console.log(fakeForEach(['tincho', 'peter', 'nico'], x => x +'Medialunas'));

Se recorre el array provista y se ejecuta la funcion de concatenar el elemento del array con el string "Medialunas"

Multiplicar los elementos del array

console.log(fakeForEach([1, 2, 3], multiplyBy2 => multiplyBy2 * 2));
